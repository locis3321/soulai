PART 02.B · Marketplace 删除决策 · admin 端落地

v0.3「Marketplace 不引入第三方内容」是用户端核心决策。admin 当前 MarketplacePage.tsx 35KB（最大单页之一）仍在维护 practitioners（第三方命理师入驻 / 评论 / 异常检测 / 投诉处理）全流程。

决策：该页面在 admin v1.0 中标注为「已下线 · v0.3 决策」并加红色横幅，路由保留但显示下线说明；历史数据（practitioners / reviews / bookings / disputes）迁移到 ai-masters 作为「5 位大师 + 历史第三方入驻数据」双视图；不再开发 marketplace 新功能（如 weight 算法、anomaly detect、review flag 等不再迭代）。

社区页面改名为「氛围运营」：保留用户动态 / 官方公告 / 内容审核；删除 marketplace 行为（用户不能在该页面发布服务、不能下单第三方）；与 v0.3「无第三方命理师入驻」决策一致。
